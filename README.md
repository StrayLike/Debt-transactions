# Проєкт "Облік позик" (Loan Tracker)

Консольна утиліта для ведення обліку фінансових заборгованостей між фізичними особами. Програма дозволяє фіксувати позики, відстежувати статус їх повернення та розраховувати чистий баланс взаєморозрахунків.

## Основні можливості
* **Облік позик:** додавання записів про те, хто, кому і скільки винен.
* **Керування статусами:** можливість позначити борг як "повернутий".
* **Баланс:** миттєвий підрахунок чистого фінансового балансу для будь-якої особи.
* **Фільтрація та сортування:** гнучкий пошук записів за особою, статусом та датою.
* **Експорт даних:** вивантаження бази позик у формати CSV та JSON.
* **Імпорт:** можливість завантаження даних з інших JSON-файлів.
* **Візуалізація:** зручна таблична верстка з використанням кольорового кодування (ANSI).

## Системні вимоги
* Python 3.10 або вище.
* Програма використовує лише стандартні бібліотеки: `json`, `os`, `csv`, `datetime`.
